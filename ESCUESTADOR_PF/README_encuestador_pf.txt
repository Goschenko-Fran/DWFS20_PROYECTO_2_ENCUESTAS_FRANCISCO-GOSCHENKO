📝 Sistema de Encuestas (Programación Funcional)

💡 Descripción:
Este código permite crear encuestas, registrar votos y mostrar resultados de forma sencilla en la consola. Cada encuesta puede tener máximo 8 preguntas, y cada pregunta máximo 3 respuestas.

⚡ Características

🆕 Crear encuestas con límite de preguntas y respuestas.

✅ Registrar votos por pregunta y respuesta.

📊 Mostrar resultados de todas las encuestas.

🏆 Mostrar la opción ganadora de cada pregunta.

⚠️ Alertas automáticas si se exceden los límites o si se intenta votar en opciones inexistentes.

📌 Ejemplo de Uso: En archivo Js.


📂 Almacenamiento

Todas las encuestas se guardan en un arreglo listaDeEncuestas durante la ejecución del programa.

🛠 Requisitos

Navegador moderno con consola abierta (F12)

JavaScript habilitado
